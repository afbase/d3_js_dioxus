Your goal is to make a wrapper over the d3.js library using dioxus-use-js crate.

You must use either: 
- https://cdn.jsdelivr.net/npm/d3@7.9.0/dist/d3.min.js
- https://cdn.jsdelivr.net/npm/d3@7.9.0/dist/d3.js

As the reference d3.js.  Feel free to use the BinBuild type in  dioxus-use-js in a build.rs if it makes sense.
And then make an example using the d3-delaunay library that is in d3.js or d3.min.js so as an example.
